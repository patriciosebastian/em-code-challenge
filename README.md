## What It Does

Everything that V1 does, plus:

1. Store each brewery's 'Likes' and 'Dislikes' in a SQLite 3 database.
2. In a separate page, displays the top ten breweries based on the formula of 'Likes' - 'Dislikes'.

## Tech Used

- HTML
- Tailwind CSS
- daisyUI
- Vue.js 3
- Vue Router
- PHP 8.2
- Laravel 10
- SQLite 3
